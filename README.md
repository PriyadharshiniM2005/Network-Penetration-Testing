# Network Penetration Testing

This repository contains structured documentation of my learning and hands-on practice in **Network Penetration Testing**. 
It covers methodologies, reconnaissance, scanning, enumeration, vulnerability assessment, and exploitation concepts used by ethical hackers to secure network infrastructures.

---

## Key Topics Covered

- Introduction to Network Penetration Testing
- Internal vs External Pentesting
- Rules of Engagement (ROE), Scope & NDA
- Black-box, Gray-box & White-box Testing
- PTES Methodology
- Reconnaissance & OSINT
- Network Scanning & Service Enumeration
- Vulnerability Scanning with Nessus
- Exploitation Case Studies
- Credential-based Attacks (Conceptual)
